# 🎓 AI Study Risk Predictor

An end-to-end Machine Learning web application that predicts a student's academic risk level based on study habits and provides personalized recommendations for improvement.

## 🚀 Project Overview
Many students struggle to identify whether their daily habits are helping or hurting their academic performance.  
This project analyzes important study-related factors and predicts whether a student is at:

- 🟢 Low Risk
- 🟡 Medium Risk
- 🔴 High Risk

The system also provides:
- Personalized advice
- Motivation messages
- Visual risk meter

This helps students improve their routine before performance drops.

---

## ✨ Features
✔ Beautiful and user-friendly web interface  
✔ Predicts academic risk using Machine Learning  
✔ Real-time input using sliders  
✔ Personalized recommendations  
✔ Motivation messages based on prediction  
✔ Visual risk meter  

---

## 🧠 Machine Learning Pipeline
1. Created custom dataset  
2. Performed preprocessing  
3. Split dataset into train/test sets  
4. Trained model using **Random Forest Classifier**  
5. Saved model for prediction  
6. Connected ML model with Flask frontend  

---

## 📊 Input Features
The prediction is based on:

- Study Hours
- Attendance (%)
- Sleep Hours
- Assignment Completion (%)
- Quiz Marks
- Screen Time
- Participation Level
